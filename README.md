# AI Options Trading Advisor

Торговый советник по опционам на бирже **Bybit**: собирает рыночные данные в реальном времени, анализирует волатильность и греки, подсказывает моменты входа в конструкции **стрэнгл / стрэддл** и направленные сделки. Управление и уведомления — через **Telegram**.

> Учебный / исследовательский проект. Не является инвестиционной рекомендацией. Торговля опционами сопряжена с высоким риском.

## Для кого

Для трейдера, который:

- торгует опционами BTC, ETH, SOL на Bybit;
- ищет входы, когда **ask Call и Put** по одному страйку и экспирации **сходятся по цене** (относительная разница обычно менее 1%);
- хочет опираться на **IV Rank**, греки, GEX и историю котировок, а не только на визуальную оценку;
- получает сигналы и статус в Telegram, без постоянного нахождения у терминала.

## Что умеет система

### Мониторинг и сигналы (Telegram-бот)

- Подписка на опционы по базовому активу, дате экспирации и страйку.
- Поток котировок с Bybit по WebSocket.
- **Сигнал входа:** Call и Put «сравнялись» по ask (порог настраивается).
- **Сигнал сброса:** цены разошлись после равенства.
- Просмотр списка опционов, текущих цен, IV и греков.

### Аналитика и LLM-агент

- Сохранение истории в **SQLite** (снимки каждые 5 минут).
- Расчёт **IV Rank**, процентилей IV, трендов греков.
- Фильтры по волатильности, аномалии объёма и OI, **GEX** (гамма-экспозиция).
- Периодический анализ рынка через **DeepSeek** с формированием торговых гипотез и чек-листа входа.

### Веб и инфраструктура

- **API-сервис** — текущие котировки и подписки для других сервисов.
- **Сервис мониторинга** — фоновая проверка пар Call/Put.
- **Админ-панель** (`static/admin.html`) — логи и состояние системы.
- Запуск в **Docker Compose** (три контейнера).

## Быстрый старт

### Требования

- Docker и Docker Compose
- Аккаунт Bybit (для рыночных данных; API-ключи — по необходимости)
- Telegram-бот ([@BotFather](https://t.me/BotFather))
- Ключ DeepSeek API (для LLM-агента)

### Настройка

1. Клонировать репозиторий.
2. Скопировать шаблон секретов:

   ```bash
   cp .env.example .env
   ```

3. Заполнить `.env` (файл **не коммитится**):

   - `TELEGRAM_TOKEN`
   - `BYBIT_API_KEY` / `BYBIT_API_SECRET` (опционально)
   - `DEEPSEEK_API_KEY`

4. Запуск:

   ```bash
   docker compose up --build -d
   ```

### Порты

| Порт | Сервис |
|------|--------|
| 8000 | API (котировки) |
| 8001 | Мониторинг |

На продакшене порты лучше не открывать в интернет без VPN или reverse proxy.

## Логика сигнала «равенство Call/Put»

Для пары опционов с одним underlying, экспирацией и страйком:

- берутся **ask** Call и Put;
- считается относительная разница к средней цене;
- если разница **меньше порога** (по умолчанию около 1%) — фиксируется **равенство** (кандидат на стрэнгл/стрэддл);
- при расхождении после равенства — **сброс** сигнала.
