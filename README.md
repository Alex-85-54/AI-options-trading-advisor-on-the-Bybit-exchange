# option_ai_bot
Options analytics using LLM, with Telegram bot interface
# scripts/fix_is_otm.py
Скрипт, который устанавливает:
- правильно рассчитанный флаг к колонке is_otm
- создает колонку option_type(если не создана) и проставляет значения P и С
- создает и проставляет значения колонки strike 
При запуске с флагом --dry-run в БД ничего не пишется (ни новые колонки, ни обновления), только выводится статистика.
